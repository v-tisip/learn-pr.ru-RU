Чтобы решить проблему хранилища для онлайн-портала обучения, вы решаете хранить данные в облаке. Но вы беспокоитесь о безопасности, резервном копировании и аварийном восстановлении в облачном хранилище. Вам кажется, что будет сложно управлять данными, размещенными в облаке.

## <a name="what-is-azure-storage"></a>Что такое хранилище Azure?

Служба хранилища Azure — это облачное решение, которое предоставляет хранилище с высоким уровнем безопасности и масштабируемости для нужд современного бизнеса. Его расширенные возможности решают все проблемы облачного хранилища и предоставляют вам надежное и устойчивое решение для хранения.

## <a name="benefits-of-using-azure-to-store-data"></a>Преимущества использования Azure для хранения данных

![Преимущества](../images/Benefits.png)

Ниже приведены некоторые из преимуществ использования службы хранилища Azure.

- **Автоматическое резервное копирование и восстановление** — эта функция снижает риск потери данных при непредвиденных сбоях.
- **Репликация по всему миру** — служба хранилища Azure реплицирует копии ваших данных, чтобы защитить их от запланированных или незапланированных событий, например планового технического обслуживания или сбоя оборудования. Вы можете реплицировать данные в нескольких расположениях по всему миру.
- **Поддержка анализа данных** — служба хранилища Azure поддерживает возможности анализа потребления данных.
- **Шифрование** — служба хранилища Azure шифрует все данные для надежной защиты. Она строго контролирует доступ к данным.
- **Поддержка нескольких типов данных** — Azure хранит данные почти любого типа. Она может обрабатывать видеофайлы, текстовые файлы и даже большие двоичные файлы, такие как виртуальные жесткие диски. Она также предлагает различные возможности для реляционных данных и данных NoSQL.
- **Поддержка структурированных и неструктурированных данных** — хранилище BLOB-объектов Azure и Azure Cosmos DB позволяют хранить как структурированные, так и неструктурированные данные.
- **Хранение данных на виртуальных дисках** — Azure также может хранить до 8 ТБ данных на виртуальных дисках. Это очень удобно при хранении больших объемов данных, например видео и симуляций.
- **Уровни хранилища** — служба хранилища Azure имеет несколько уровней для доступа к данным в зависимости от частоты их использования. Это значительно улучшает взаимодействие с пользователем.
